Data Kitchen (jz-fork)

  1. Download the Zipfile for your OS

    • Linux
    • Mac
    • Windows
  2. Unzip the Zipfile anywhere and then change into the data-kitchen folder
  3. Run the data-kitchen executable (e.g. ./data-kitchen)
  4. You may edit the config file located on linux in the data-kitchen/resources/app/config.json and in Mac in the data-kitchen.app/Contents/Resources/app/config.json. See the in-app Help for details

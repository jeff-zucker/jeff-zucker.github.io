Data Kitchen (jz-fork)

  • 1. Download the Zipfile for your OS
    • Linux x64
    • Mac/Darwin
    • Windows
  • 2. Unzip the Zipfile anywhere and then change into the data-kitchen folder
  • 3. Run the data-kitchen executable (e.g. ./data-kitchen)
  • Note: You may edit the config file is located in the data-kitchen/resources/app/config.json. (Turtle version coming soon)
